Este directorio contiene los siguientes archivos:

listaLigada.c
listaLigada.h
Makefile
Readme.txt
usaLista.c


En los archivos listaLigada.c y listaLigada.h se definen las listas ligadas y sus métodos asociados.

El archivo usaLista.c es un programa de ejemplo que importa las definiciones de lista ligada y las utiliza.

El archivo Makefile sirve para automatizar la compilación y limpieza del programa (para sistemas tipo UNIX). Al correr el comando make genera un ejecutable llamado usaLista.

El Readme.txt es el archivo que está leyendo.
